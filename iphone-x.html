FAQs About iPhone X Repair

Where's the iPhone X home button?

The iPhone X is the first iPhone to be designed without a home button. The Touch ID is no longer used to unlock the device. Instead, the front camera is used for Face ID, which is similar to using a fingeprint to unlock the device. Instead of pushing the home button to get to the home screen, the iPhone now uses a swipe upwards for the same functionality. The iPhone X is now all display, bringing a better user experience.
